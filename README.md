# Smart Task Queue System

A production-grade, event-driven job processing platform built with Java Spring Boot, Kafka, PostgreSQL, and Redis.

---

## Tech Stack

| Technology | Purpose |
|---|---|
| Java 17 + Spring Boot 3.2 | Core backend framework |
| PostgreSQL | Persistent job storage |
| Kafka | Async event streaming between services |
| Redis | Idempotency control + caching |
| Python FastAPI | AI job classification service |
| Docker Compose | Local orchestration of all services |

---

## Project Roadmap

- **Week 1** — Core Foundation (Kafka + PostgreSQL + Redis + Reliability)
- **Week 2** — Security + Multi-Tenancy + AI Integration
- **Week 3** — Observability + Kubernetes + Deployment
